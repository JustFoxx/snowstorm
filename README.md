Creates snow storm in your command line

## Usage
Usage: snowstorm [OPTIONS]

Options:
  -d, --distribution <DISTRIBUTION>  [default: 50]
  -S, --speed <SPEED>                [default: 1000]
  -s, --snowflake <SNOWFLAKE>        [default: *]
  -a, --air <AIR>                    [default: " "]
  -B, --space-between                
  -h, --help                         Print help
  -V, --version                      Print version
